# BitDust Desktop

A repo for generic BitDust launcher on different desktop platforms


## Install dependencies

    npm install


## Start in "developer mode"

    npm start dev


## Build for Windows platforms

    make build_exe


## Build for MacOSx

    make build_dmg


## Build for Linux Debian

    make build_deb
